## Highlights

- First public release of SEAJelly
- Serverless multi-channel AI agent platform with setup wizard and admin dashboard
- Built-in self-evolution workflow through GitHub and Vercel
- Weighted multi-key LLM routing for better concurrency and resilience
- Sub-Apps, coding sandboxes, voice, skills, MCP, knowledge base, and subscriptions

## Setup

- Recommended install path: Vercel one-click deploy, then open `/setup`
- Setup validates bootstrap environment variables before running SQL
- Production setup may show a dedicated security login URL that should be saved immediately

## Notes

- This release establishes the `v0.1.0` baseline for future one-click upgrades
- Future releases should include `.seajelly/upgrade-manifest.json`
- If a future release includes database changes, it should also include a SQL file referenced by the manifest
