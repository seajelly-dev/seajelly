## Changes

- Formalized `v0.1.7` as the single long-term baseline for future one-click upgrades
- Marked the temporary bridge line around `v0.1.6` as withdrawn so the updater no longer treats it as an officially supported path
- Improved upgrade-path messaging for installs that are already on a withdrawn release line

## Notes

- Future official one-click upgrades now continue from `v0.1.7`
- Older recovery lines such as `v0.1.4` and `v0.1.6` are no longer part of the maintained upgrade path
- Installs still on a withdrawn line should recover by redeploying from the current `stable` branch
