## Changes

- Added Weixin channel support across sender adapters, webhooks, dashboard platform surfaces, and the edge-gateway longpoll bridge
- Added `tools/weixin-login/login.mjs` plus gateway install/docs to help bring up the Weixin bridge
- Improved knowledge search so it reads the embedding model from system settings instead of hard-coding it
- Tightened channel access handling when inbound events arrive without a bound channel, with tests covering approval and open modes

## Notes

- This release includes a database update so existing installs can store `weixin` in `channels.platform` and `events.source`
- After the code deploy reaches `READY`, apply the database step in `Dashboard -> Updates` to complete the upgrade
